# Transportation & Delivery Tracking System

A full-stack Flask-based web application for managing shipment tracking and delivery status updates.

## Features
- Admin Login
- Add Shipment
- Edit Shipment
- Delete Shipment
- Search Shipments
- Dashboard Analytics
- SQLite Database
- Responsive UI

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Git & GitHub

## Run Project

```bash
pip install -r requirements.txt
python app.py
